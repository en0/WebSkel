SKEL Angularjs/Bootstrap

This is a skeleton for angularjs with bootstrap under python-flask.

The following files are refrenced from the root of web application

/templates/index.html
The site template for angularjs. Loads angular, bootstrap and site files
/templates/partials/*
All the partials for the angularjs views
/static/js/*
Site specific javascript files
/static/js/ui-bootstrap-tpls-0.11.2.min.js
Bootstrap components for angularjs
/static/css/site.css
Site wide CSS files (empty).

Since Angular and flask use the same layout variable syntax. ie: 

{{ "{{ VARIABLE }}" }}
The Angularjs interpolation has been modified to: 
[[ StringConst ]]
If you wish to use a diffrent delimiter, you can modify the character in /static/js/app.js
